Projetos

Ca'Med

A aplicação Ca'Med é um site e aplicativo desenvolvido com o objetivo de simplificar a busca por medicamentos. Com base nos sintomas informados pelo usuário, o sistema gera uma lista de medicamentos relacionados que podem ser indicados para o tratamento desses sintomas. A plataforma visa proporcionar comodidade e praticidade aos usuários, oferecendo uma ferramenta eficiente para encontrar os medicamentos adequados de forma rápida e segura.

SATD-ES

A aplicação SATD-ES é um sistema de aquisição de dados agrometeorológicos do Espírito Santo. O objetivo principal do sistema é coletar e disponibilizar informações meteorológicas relevantes para a agricultura e setores relacionados. Através de uma interface web, os usuários têm acesso a gráficos e tabelas que apresentam os dados coletados, como temperatura, umidade, precipitação, entre outros parâmetros. Essas informações auxiliam na tomada de decisões estratégicas para atividades agrícolas, permitindo melhor planejamento e gerenciamento. O SATD-ES visa fornecer dados precisos e atualizados, contribuindo para o desenvolvimento sustentável do setor agropecuário no Espírito Santo.

Separador de Tampinhas por Cores

O projeto em desenvolvimento consiste em um separador de tampinhas por cor, com o intuito de apoiar a ONG Pramia, uma organização dedicada a ajudar animais em situação de rua no Espírito Santo. O objetivo principal é facilitar a separação das tampinhas por cores específicas, otimizando o processo de coleta e triagem. As tampinhas arrecadadas serão destinadas à ONG, que utilizará os recursos para realizar ações como castração, adoção responsável, cuidados veterinários e alimentação dos animais resgatados. O projeto busca contribuir para a causa animal e o bem-estar dos animais de rua, promovendo a conscientização e o engajamento da comunidade em prol dessa importante causa.
